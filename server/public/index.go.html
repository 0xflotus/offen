Welcome to offen web analytics

View and manage the usage data this website has collected from you.

Open auditorium

offen is a free and open source analytics software for websites that allows respectful handling of data. It is installed on the website that linked you here.

Below will find answers to the most common questions website users have.

How can I view and manage my data?

Go to the auditorium.

What data is collected?

Only page views, sessions, referrer and visited URL are collected. From this, other metrics such as the bounce rate are calculated.

What data isn't collected?

Location data, device information and engagement are not collected. There are also no fingerprinting technologies in use.

How is my data handled?

Your data is encrypted end-to-end. It will be deleted after 6 months at the latest. offen does not share your data with third-parties.

Can I delete my data?

Yes you can delete your data completely at any time on all participating websites. Go to the auditorium.

How does it all work?

After opening a website that has offen installed you can explicitly opt in to the data collection and thereby help to improve the services you use.

Only then will you be assigned an ID using a cookie. Learn more about cookies here.

offen handles this ID in an unrecognizable form. Operators can identify you only within one website.

Visit www.offen.dev for more information.
